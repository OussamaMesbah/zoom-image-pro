# 🔍 Zoom Image Pro

**Zoom Image Pro** is a high-performance Chrome extension designed for seamless image interaction. Whether you're a designer, photographer, or just browsing, it provides a powerful set of tools to zoom, pan, rotate, and save images on any website.

---

## 🌟 Key Features

*   **Smart Google Images Support:** Automatically detects and upgrades blurry thumbnails to high-resolution preview images.
*   **Dual Zoom Styles:** Choose between classic **Fullscreen Zoom** or a circular **Magnifying Glass (Loupe)**.
*   **Advanced Image Control:** Rotate (90° steps), Flip (Horizontal/Vertical), and Fit-to-Screen instantly.
*   **Gallery Navigation:** Quickly browse through all images on a page using `[` and `]` keys.
*   **Built-in Security:** Sanitizes SVGs and validates image URLs to ensure a safe browsing experience.
*   **Context Toolbar:** Right-click any zoomed image for quick access to "Search by Image", "Copy Link", and "Save".
*   **Image History:** Keep track of your last 50 zoomed images in a private local log.

---

## 🕹️ Activation Modes

You can switch between these modes in the extension settings:

### 1. Hold Mode (Default)
*   **How it works:** Hold your trigger key (default: `Alt`) and scroll the mouse wheel over an image to start zooming.
*   **Exit:** Simply release the trigger key to close the zoom.
*   **Context Menu:** If you right-click to open the toolbar, the zoom will stay open even if you release the key.

### 2. Snap Mode
*   **How it works:** Simply scroll over an image to "lock" the zoom. It stays active without holding any keys.
*   **Exit:** Click anywhere, press `Esc`, or scroll back to 1x scale to close.

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
| :--- | :--- |
| **Alt + Scroll** | Zoom In / Out (Hold Mode) |
| **Mouse Drag** | Pan Image |
| **`S`** | Download / Save Image |
| **`R`** | Rotate 90° |
| **`H` / `V`** | Flip Horizontal / Vertical |
| **`F`** | Fit to Screen |
| **`[` / `]`** | Previous / Next Image in Gallery |
| **`0`** | Reset View |
| **`Esc`** | Close Zoom |

## 🛠️ Installation (Developer Mode)

Since this extension is hosted on GitHub, you can install it manually:

1.  **Download** the latest [release ZIP](https://github.com/OussamaMesbah/zoom-image-pro/releases).
2.  **Extract** the ZIP file to a folder on your computer.
3.  Open Chrome and go to `chrome://extensions/`.
4.  Enable **"Developer mode"** (toggle in the top right).
5.  Click **"Load unpacked"** and select the folder where you extracted the files.
